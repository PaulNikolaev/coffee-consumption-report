# Coffee Consumption Report

CLI-проект для построения отчетов по данным о потреблении кофе из CSV-файлов.

## Структура проекта

- `main.py` - тонкая точка входа для локального запуска через `python main.py`
- `src/coffee_report/` - основной пакет приложения
- `data/` - входные CSV-файлы и примеры
- `tests/` - автоматические тесты
